## Key Python libraries

| Library | Use | Install |
|---------|-----|---------|
| **RDKit** | Cheminformatics swiss-army knife | `conda install -c conda-forge rdkit` |
| **DeepChem** | ML for drug discovery | `pip install deepchem` |
| **MDAnalysis** | MD trajectory analysis | `pip install MDAnalysis` |
| **OpenMM** | GPU MD simulations | `conda install -c conda-forge openmm` |
| **py3Dmol** | 3D molecular visualization | `pip install py3Dmol` |
| **LangChain** | LLM application framework | `pip install langchain` |
| **LangGraph** | Agentic AI workflows | `pip install langgraph` |
| **biopython** | Bioinformatics utilities | `pip install biopython` |
| **pydeseq2** | DESeq2 in Python | `pip install pydeseq2` |

## Learning resources

### Computational drug discovery
- [TeachOpenCADD](https://github.com/volkamerlab/teachopencadd) — Jupyter notebook tutorials on CADD
- [Practical Cheminformatics](https://practicalcheminformatics.blogspot.com/) — Pat Walters' excellent blog
- [SILCS documentation](https://silcsbio.com/) — Official SILCS methodology resources

### Agentic AI & LLMs
- [LangChain docs](https://python.langchain.com/)
- [LangGraph documentation](https://langchain-ai.github.io/langgraph/)
- [DeepLearning.AI short courses](https://www.deeplearning.ai/short-courses/) — Agentic AI, RAG, tool-use

### Bioinformatics
- [Bioconductor workflows](https://www.bioconductor.org/help/workflows/)
- [DESeq2 vignette](https://bioconductor.org/packages/release/bioc/vignettes/DESeq2/inst/doc/DESeq2.html)
- [Harvard Chan Bioinformatics Core training](https://hbctraining.github.io/main/)

### Molecular dynamics
- [CHARMM-GUI](https://www.charmm-gui.org/) — System builder for MD simulations
- [GROMACS tutorials](http://www.mdtutorials.com/gmx/)
- [OpenMM documentation](https://openmm.org/)
